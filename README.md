# Ludopedia-library-about-games
Ludopedia — это мобильное Android-приложение, представляющее собой интерактивную энциклопедию видеоигр. Пользователь может искать игры по названию, жанру и платформе, а также просматривать рейтинги и описания. Проект ориентирован на создание удобного, эстетичного и атмосферного справочника для геймеров.
